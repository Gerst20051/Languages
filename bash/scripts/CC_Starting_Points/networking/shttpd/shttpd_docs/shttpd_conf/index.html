Globals

DEFAULTDOMAIN

The default domain and port (used for HTTP/1.0 requests).

HAS_NEW_NETCAT

Set to 1 if your netcat is properly patched.

NC

The location of a patched nc binary from netcat

PERL_EXECUTABLE

The path to the Perl interpreter (used for CGI).

PHP_CGI_EXECUTABLE

The path to the php-cgi interpreter (must be compiled with the CGI SAPI.

REALPATH

The location of a realpath binary

SHTTPD_PORTS

A list of ports to listen on.

DEFAULTDOMAIN

The default domain and port (used for HTTP/1.0 requests). 

DEFAULTDOMAIN="127.0.0.1:8081"

HAS_NEW_NETCAT

Set to 1 if your netcat is properly patched. 

HAS_NEW_NETCAT=1

NC

The location of a patched nc binary from netcat 

NC=/usr/local/bin/nc

PERL_EXECUTABLE

The path to the Perl interpreter (used for CGI). 

PERL_EXECUTABLE="/usr/bin/perl"

PHP_CGI_EXECUTABLE

The path to the php-cgi interpreter (must be compiled with the CGI SAPI. 

PHP_CGI_EXECUTABLE="/usr/local/php5-cgi/bin/php-cgi"

REALPATH

The location of a realpath binary 

REALPATH="/usr/local/bin/realpath"

SHTTPD_PORTS

A list of ports to listen on. 

SHTTPD_PORTS="8081"
