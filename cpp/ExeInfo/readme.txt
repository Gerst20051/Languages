Description
===========
The ExeInfo utility shows general information about executable files (*.exe), dynamic-link libraries (*.dll),
ocx files, and drivers files. It can recognize all major types of executables, including
MS-DOS files, New Executable files (16-bit) and Portable Executable files (32-bit).

Using ExeInfo
=============
The ExeInfo utility doesn't require any installation process.
Just copy the exeinfo.exe to any folder you want and run it. After you run it, click the
"Show File Info" button, and select the desired executable file. After you select the file,
the ExeInfo utility will show extensive information about your executable file.
You can also get information of a file by dragging it from explorer window into the ExeInfo
window.
