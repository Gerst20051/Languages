JavaScript compression tester

This sample code uses PHP and the online Closure Compiler REST API to merge and compress JavaScript code before it is returned to the browser. The PHP code is contained in script/index.php.

IMPORTANT! This is not production-level code. Please refer to this SitePoint article for more information.
