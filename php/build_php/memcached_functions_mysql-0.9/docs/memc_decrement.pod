=head1 NAME

memc_decrement

=head1 UDF 

Libmemcached User Defined Function for MySQL 

=head1 SYNOPSIS

=head1 DESCRIPTION

memc_decrement() is a UDF used to decrement an integer value stored in 
memcached server from within MySQL. You first argument you must pass a 
key value for the integer stored. The second argument can be can either
pass a data value to decrement by, or default is 1.

=head1 RETURN

NULL

=head1 USAGE 

memc_decrement('keyfoo') <-- decrements by 1
memc_decrement('keyfoo', 3) <-- decrements by 3
