=head1 NAME

memc_behavior_set

=head1 UDF 

Libmemcached User Defined Function for MySQL 

=head1 SYNOPSIS

=head1 DESCRIPTION

memc_behavior_get() is a UDF used to retrieve the behavior of a connection
to a memcached server from within MySQL. An alias to this function is memc_servers_behavior_get();

=head1 RETURN

NULL

=head1 USAGE 

select memc_behavior_get('MEMCACHED_BEHAVIOR_SUPPORT_CAS');

or with SET:

set @setval = memc_behavior_get('MEMCACHED_BEHAVIOR_SUPPORT_CAS');
select @setval;
