Accelerometer Sample
9/9/2010

This sample shows you how to use the accelerometer to determine the orientation and motion of the Windows Phone.

You will need to install Windows Phone Developer Tools to run this sample. To get started, go to the Windows Phone developer portal.

For more information about the developing accelerometer-based Windows Phone applications see Accelerometer Overview for Windows Phone in the MSDN Library.

The current release does not include an accelerometer data emulator. To run this sample, you must have a physical device.
