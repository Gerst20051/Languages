Weather Forecast Web Service Sample
9/9/2010

This sample shows how you can call and process data from a Web Service in a Windows Phone application. The sample calls a web service to get weather forecast information, parses the returned XML, and displays the information in a Weather Forecast page.

You must install Windows Phone Developer Tools to run this sample. To get started, go to the Windows Phone developer portal.

For more information about developing Windows Phone applications that use Web Services, see Web Services for Windows Phone in the MSDN Library.
