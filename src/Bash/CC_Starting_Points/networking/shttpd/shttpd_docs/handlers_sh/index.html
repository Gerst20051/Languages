closeConnection

Instructs the netcat process to close the connection to the client. 

closeConnection()
Parameters
FD

The file descriptor for sending data to nc.

FIFO

The FIFO used for sending data to nc.

Discussion

Do not close the connection if the Content-Length status is set. Otherwise, you may end up closing the connection on the wrong client.

getStatusText

Prints a status string for a status code number. 

getStatusText()
Parameters
CODE

The HTTP status code number.

Return Value

Returns s status code string through stderr for the HTTP status code in CODE. This function is not a complete list of status codes. It only includes strings for result codes that shttpd.sh actually generates internally.

handleDirListing

Prints a directory listing. 

handleDirListing()
Parameters
FD

The file descriptor for sending data to nc.

URL

The original URL as received from the client.

LOCALDIR

The complete local path to the requested directory.

Discussion

This function is callsed by handleGetPostRequest. Before calling this function, be certain that the directory request is safe. This function does no validation whatsoever.

splitForScript

Searches for a CGI script within a relative path. 

splitForScript()
Parameters
RELATIVEPATH

The path relative to the document root.

BASEDIR

The document root.

Return Value

If it finds a script, the first part of the path (up to and including the script name) is stored in the global variable SCRIPT_PATH and the rest of the path after the script name is stored into the global variable SCRIPT_REMAINDER. If it fails to find a script in the path,

Discussion

This searches for a CGI script when it is passed in the middle of a URL. For example, "http://example.org/script.cgi/foo/bar".
