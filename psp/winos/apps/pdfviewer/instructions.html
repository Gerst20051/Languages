PDF Viewer Instructions Return To Main Page
1) To start viewing your PDF's, just click "View PDF's" on the main page.
2) Now it will take you to the Viewer screen. All you have to do is select your PDF using the "Select" button.
3) After you're done picking your PDF to view, select if you want the PDF format in either "HTML" or "Text". If you select HTML, it will come out like a webpage. If you select Text, you can save it in text format (txt) to your memory stick, to view on the go, or if it's a big file and you would like to read it while on a trip.
4) Now that you've finished picking the PDF format, click "View" to view your PDF.
